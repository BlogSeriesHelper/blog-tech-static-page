最大公约数，greatest common divisor，GCD

辗转相除法（又称欧几里得算法）

def gcd(a, b):
  return gcd(b, a % b) if b > 0 else a
  
def gcd(a, b):
  while (a % b):
    a, b = b, a % b
  return a

更相减损术

def gcd(a, b):
  while (a !=b):
    if a > b:
      a -= b
    else:
      b -= a
  return a
  
def gcd(a, b):
    if a > b:
      a -= b
    else:
      b -= a
  return gcd(a, b)

nums = [32,16,12,24,36,20,40,28]
res = 0 # 最大公约数初始值可以设置为0，或者取第一个数 gcd(0, x) == x
for num in nums:
  res = gcd(res, num)

最小公倍数，Least Common Multiple，lcm

两个数的最小公倍数等于两数之积除以两数最大公约数。

def lcm(a, b):
  return a // gcd(a, b) * b

多个数的最小公倍数，求前两个数的最小公倍数然后，让前两个数的最小公倍数再与第三个数求最小公倍数，以此类推，当然，也是可以设置最小公倍数默认值为1

nums = [40,20,10,30, 25, 90]
res = 1 # gcd(1, x) == x
for num in nums:
  res = lcm(res, num)

Python中有库函数math.gcd、math.lcm
